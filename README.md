# 🛒 Online Store

A modern, responsive e-commerce frontend built using React, Tailwind CSS, and Vite. This project demonstrates a full shopping workflow including product browsing, cart management, and checkout system.

---

## 🚀 Live Demo

👉 https://your-vercel-link.vercel.app

---

## ✨ Features

* 🛍️ Product Listing with Search & Filters
* 📦 Product Detail Page
* 🛒 Cart Management (Add / Remove / Quantity)
* 💳 Checkout System
* 🔐 Authentication UI (Login / Register)
* 🌙 Dark / Light Mode Toggle
* ⚡ Fast Performance (Vite + Lazy Loading)
* 🔔 Toast Notifications
* 📱 Fully Responsive UI

---

## 🧠 Tech Stack

* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS (v4)
* **Routing:** React Router
* **State Management:** Context API
* **Icons:** Lucide React
* **Notifications:** React Toastify
